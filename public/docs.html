Dokumentasi Sylent AI

Sylent AI adalah aplikasi chat AI modern yang terinspirasi dari antarmuka Claude, dibangun dengan stack ringan: HTML, CSS, Vanilla JavaScript di sisi frontend, dan Node.js + Express + Upstash Redis di sisi backend — semuanya dalam satu project.

Fitur Chat

  • Chat baru & riwayat percakapan di sidebar
  • Rename dan hapus chat
  • Render Markdown lengkap dengan syntax highlighting
  • Tombol copy kode dan copy respons
  • Auto scroll saat respons masuk
  • Indikator loading & efek typing
  • Regenerate dan stop generate
  • Enter untuk kirim, Shift+Enter untuk baris baru

Markdown & Kode

Semua respons Sylent AI dirender menggunakan Markdown renderer dan disanitasi dengan DOMPurify sebelum ditampilkan, sehingga aman dari serangan XSS. Blok kode otomatis mendapat syntax highlighting via Highlight.js.

Live Preview

Jika Sylent AI menghasilkan kode HTML, SVG, CSS, atau JavaScript, tombol Preview akan muncul otomatis di bawah blok kode. Preview dijalankan dalam iframe ter-sandbox, lengkap dengan opsi Fullscreen, Refresh, Open in New Tab, dan Download.

Model

Sylent AI saat ini hanya menyediakan satu model: Sylent 0.1 — dioptimalkan untuk percakapan umum, pembuatan kode, dan bantuan produktivitas sehari-hari.

Privasi & Keamanan

Backend menggunakan Helmet, Content Security Policy, rate limiting, dan validasi input untuk menjaga keamanan layanan. Kunci admin disimpan di Redis dan tidak pernah dikirim ke frontend.
